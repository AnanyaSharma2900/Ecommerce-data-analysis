# Ecommerce-data-analysis
## project overview
This project analyzes an e-commerce dataset to understand sales trends, product performance, and return patterns. The goal is to extract useful insights that can help in better business decision-making.

## Steps Performed

### 1. Data Cleaning
- Handled missing values
- Removed duplicates
- Standardized product prices
- Handled outliers 
- Corrected data types

### 2. Feature Engineering
- Created Total column using Price × Quantity

### 3. Exploratory Data Analysis (EDA)
- Category-wise sales analysis
- Monthly and yearly sales trends
- Product performance analysis
- Order status and return analysis
- Payment method analysis

## Key Insights
- Books category has the highest overall sales  
- Shoes is the top-selling product  
- Highest sales observed in July 2025  
- High return rate observed in Fiction category  
- Cash on Delivery is the most used payment method
